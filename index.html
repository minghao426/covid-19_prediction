Data

Data used for the Kalman filter and the SEIR model is obtained from Johns Hopkins University's CSSE GitHub database, which updates its data hourly throughout the day. The raw data format is a time-series of the confirmed cases with additional information such as locations by county, province and state. The earliest data point appears on January 22nd and each day after to the present day is recorded. In order to process the raw data, some preprocessing transformations including column combinations, and summation of cell values are performed in order to properly feed the data into our algorithmic models. The most updated set of data is used whenever the mathematical models are executed.

Methods

The Kalman model and the SEIR model are used as our methods to estimate the spread of coronavirus pandemic and further conduct future predictions.

1. Kalman Filter

The Kalman filter is a recursive mathematical process that uses a combination of equations and actual data input to estimate the true value, and in this case the number of confirmed cases and the number of deaths based on a time series analysis. The algorithm can be real-time, meaning that the estimate can be calculated based on only the present input measurements and the previously calculated state transition matrix.

2. SEIR Model

SEIR is a model to estimate the pandemic spreading trend based on population data. S, E, I, R respectively represents susceptible individuals, exposed individuals (infected but not yet infectious or symptomatic), infected individuals (severity increases with class i and we assume individuals must pass through all previous classes) and recovered individuals (now immune).

Challenges and future work

For future work, we may consider more possible relevant parameters like medical services and population movement to further perform analysis and prediction of the spread of Covid-19. From a broader perspective, we can even change our project objects from the state level to the country level and comprehensively research the main factors which have the biggest influence on the infection speed.

What’s more, for the SEIR model, the infected population can be divided into concrete type Ii, which represents different disease severity. Then, we can further analyze the relationship between each disease stage with the level of medical services according to this severity.
